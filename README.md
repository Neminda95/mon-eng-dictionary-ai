# Mon-English Dictionary AI Project 🤗

This repository is dedicated to the digitization, preservation, and advancement of the **Mon language (ဘာသာမန်)** for Artificial Intelligence (AI) and Natural Language Processing (NLP) applications. 

As a low-resource language, providing high-quality, structured open-source data is a critical milestone for integrating Mon into modern language models and AI technologies.

---

## 📊 Dataset Overview

This project provides a curated bilingual dataset containing high-quality Mon-English vocabulary pairs and detailed dictionary definitions. To support various computational linguistics tasks, the data is provided in two distinct formats:

1. **Raw Dictionary Data (`.csv`):** Cleaned and structured vocabulary mappings suitable for machine translation, dictionary applications, and linguistic benchmarking.
2. **Instruction-Prompt Dataset (`.json`):** Formatted into `Instruction-Input-Output` structures, specifically optimized for fine-tuning Large Language Models (LLMs) like ChatGPT, Llama, and Gemma to accurately understand, translate, and generate the Mon language.

---

## 📁 Repository Structure
* `mon_eng_dictionary_combined_excel_ready.csv`: The raw bilingual Mon-English dictionary dataset.
* `mon_eng_instruction_dataset.json`: The compiled instruction dataset ready for LLM fine-tuning and AI training.
